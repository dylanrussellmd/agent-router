## TL;DR

```bash
npx -y @dylanrussell/omo-router init       # one-time bootstrap
omo-router list                             # see your stacks
omo-router use openrouter-cheap             # switch to a cheaper mix
# restart opencode
omo-router back                             # change your mind
```

That's pretty much it. Everything else is for when you want more control.
